BioSigPlotFHR
   Class to visualize FHR signals. 
  It is a pre-configuration of BioSigPlot for Foetal Heart Rate.
Usage
    >> BioSigPlot(FHR,TOCO, 'key1', value1 ...);
Input
    FHR  
       Foetal Heart rate data 
 
    TOCO
       TOCO data
 
    key1, key2,... : same as BioSigPlot
