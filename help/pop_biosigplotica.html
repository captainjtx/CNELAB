pop_biosigplotica
  pop_biosigplotica() - Visualise multichannel signals and enable to easily
  switch to ICA as if it was a second Dataset. The ICA data is entirely
  computed and it can take more memory that pop_biosigplot()
Usage
    >> pop_biosigplot( EEG )
