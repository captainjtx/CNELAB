Installation
  Installation
     You just have to set the current folder of Matlab to the BioSigPlot
     directory.
     You can also add the BioSigPlot directory to the path list in (File>Set Path)
     or doing it automatically by executing the script "Installation"
         
     If you want to use it as EEGLab plugin you have to first install EEGLab
     and then copy the BioSigPlot directory into EEGLab "plugins"
     directory.
 
     You can have a look to the demonstration video on
     http://biosigplot.sourceforge.net
