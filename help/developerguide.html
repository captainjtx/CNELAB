Developer Guide
  
  This part presents the list of classes and function used by BioSigPlot
  If you want to realise additionnal developpement you can have a look on 
  those other help files.
Classes
    ConfigWindow - Window to edit BioSigPlot properties and saving it
    EventWindow  - Window used to browse the various events marks 
    ExportWindow - Windows used to configure exportation to image or PDF
    VideoWindow  - Windows to display Video thanks to VLC ActiveX
Functions
    butterfilt        - Complete butterworth filtering process
    obj2str           - Convert any object to a string that you can eval to get back the object 
    pop_biosigplotica - EEGLab function to plot ICA channels
    pop_biosigplot    - EEGLab function to plot signal
